rdo-repos
=========

A basic role to remove the base fedora repos configure the fedora RDO repos.


Role Variables
--------------

.. list-table:: Variables used for provision-node
   :widths: auto
   :header-rows: 1

   * - Name
     - Default Value
     - Description
   * - `rdo_fedora_repo`
     - current
     - rdo repo to use. Should be current or consistent
